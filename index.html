Projects

Project 1

The first project is related to develop a single Web page using only two technologies: HTML and CSS. The project consists of creating a single web page with a menu at the top know as the Navigation Bar. We have to include text describing ourself and text describing the project. Additionally, we need to include two images: one image is of ourself and second image is anything related to the project itself.

Project 2

The second project was created during a semester-long class studying software development practices. I was a frontend developer, so I designed the user interface and utilized REST API requests to access the database. My end of the project was designed on Android Studio. We also learned about GitHub version control practices. The app that my team and I created was a simple generative obstacle shooter game.

Project 3

The third project was designed during my summer internship at Vermeer Corporation. I, along with another intern, created an internal tool using Python and various libraries to access machine data. The project would filter, clean, and display said data to company stakeholders within a simple, interactive application.

Project 4

The fourth project is currently being worked on in my Computer Architecture and Assembly Level Programming class.
